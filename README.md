## TODOLIST with React + UseContext
This project was built using [GitHub Pages](https://mkremnev.github.io/todolist/)

#### Built with
- React + Hooks (UseContext)
- Material-UI
- Emotion

#### Getting Started

##### Installation
1.Clone the repo
```
    git clone https://github.com/mkremnev/todolist.git
```
2.Install NPM packages
```npm
    npm i
or
    npm install
```

#### Usage
```
    npm run start
```
